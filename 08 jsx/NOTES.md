## Concept Kya Hai? (Simple Hinglish Explanation)
- **JSX:** JSX JavaScript + markup ka mix syntax hai jo React elements create karta hai.
- **Expression in JSX:** Curly braces `{}` ke andar dynamic values ya JS expressions likhte hain.
- **Fragment:** Fragment (`<>...</>`) extra div diye bina multiple elements return karne deta hai.
