## Repository Goal

- Practice React step-by-step from beginner to advanced level
- Learn and apply each concept in an isolated folder
- Build a real project mindset: state, routing, API, optimization, architecture
- Speed up revision using notes + diagrams

## Topics Covered (05 to 53)

### Fundamentals

- Basics of React, JSX, Imports, Components, Styling
- Props, Data Flow, State, Hooks
- Event Handling, Conditional Rendering, List Rendering
- Input handling: text, checkbox, radio, dropdown

### Core React + Intermediate

- `useRef`, custom UI projects (Counter, Quiz, Password Generator)
- API calls (`fetch`, `axios`)
- React Router DOM
- Context API

### State Management

- React Redux (core flow)
- Redux Toolkit (modern Redux)
- Zustand (lightweight global state)

### UI and Libraries

- Tailwind CSS
- React Icons
- MUI + MUI project components

### Performance and Advanced Concepts

- HOC and Custom Hooks
- Lifecycle methods
- `React.memo`, `useMemo`, `useCallback`
- Optimization patterns
- `forwardRef`
- Lazy Loading (`React.lazy`, `Suspense`)
- React Virtualization

## Folder Structure Format

- Each folder represents a numbered topic/module
- Most folders include:
  - source code (`src/`)
  - concept-specific implementation
  - `NOTES.md` (concept explanation + important files + flow + diagrams where needed)

## How to Run Any Folder

```bash
cd "<folder-name>"
npm install
npm run dev
```

Example:

```bash
cd "41_Redux_Toolkit"
npm install
npm run dev
```

## Recommended Learning Order

1. `05` -> `23` (Core basics)
2. `24` -> `35` (Practice + projects + router + forms + API)
3. `36` -> `44` (Global state + UI libraries)
4. `45` -> `53` (Advanced React + performance)

## Notes and Revision

- Open `NOTES.md` for each module
- Start with the "What is the Concept?" section
- Map concepts to the listed "Important Files"
- Visualize flow in diagram-based folders (Redux, Context, Lazy Loading, Virtualization, etc.)

## Best Use Strategy

- Study one module -> run the code -> revise notes
- Add one small feature by yourself
- Explain the concept in your own words
- Then move to the next folder

If you are learning React seriously, this repo is a complete hands-on roadmap from fundamentals to performance-level concepts.
